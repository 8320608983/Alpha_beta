# Alpha_beta
 
## Overview
This is a Unity-based card matching game prototype developed as part of a technical assignment.

## Features
- Dynamic grid system (2x2, 2x3, 4x4, 5x6)
- Smooth card flip animations
- Queue-based matching system (non-blocking input)
- Score and combo system
- Game over panel with restart
- Basic save system (grid persistence)
- Sound effects (flip, match, mismatch, game over)

## Controls
- Click on cards to flip and match pairs

## Tech
- Unity 2022 LTS
- C#

## Notes
The focus of this project was on gameplay logic, clean architecture, and performance optimization.
